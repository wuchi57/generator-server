# 基于yeoman创建的项目模版

#### 项目技术栈包含：express、dotenv、body-parser

#### 使用方式
- 全局安装依赖: yarn install -g yo generator-server
- 查看yo已安装的项目模版：yo --generators
- 删除项目模版：npm uninstall generator-server -g
- 使用项目模版：mkdir test && cd test && yo server && node app
